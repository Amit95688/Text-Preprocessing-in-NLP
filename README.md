# NLP Text Preprocessing Notebook

This repo contains a Jupyter notebook with step-by-step theory and code for common text preprocessing tasks in NLP:

- Lower casing
- Removing punctuation
- Removing stopwords
- Removing URLs & HTML tags
- Chat word conversion
- Spelling correction
- Emoji handling
- Tokenization & Lemmatization

## Usage
Open `NLP.ipynb` and follow the sections. Run cells to process your own text data.

## Requirements
Python 3, pandas, numpy, nltk, emoji, textblob

Install with:
```
pip install pandas numpy nltk emoji textblob
```
